# Wishlist de Photocards do Twice

Sobre o projeto:

Este repositório contém a minha wishlist pessoal de photocards, criada com o objetivo de organizar e acompanhar a evolução da minha coleção física de forma digital.
O projeto foi desenvolvido totalmente voltado para uso próprio, exatamente do jeito que eu sempre quis: visualmente agradável, simples de usar e focado naquilo que realmente importa pra mim — saber o que eu já tenho e o que ainda falta.

## Funcionalidades ✅

- Organização dos photocards por era/conceito
- Marcação de status: "Tenho" e "Falta"
- Interface visual inspirada no primeiro full álbum do grupo
- Acompanhamento da coleção de forma prática pelo computador

## Observações importantes ⚠️

Este projeto é exclusivamente para uso pessoal, não sendo pensado para uso público ou colaborativo
Atualmente, o JavaScript permite que qualquer pessoa altere os botões de "Tenho" e "Falta", o que não é ideal
Isso será ajustado futuramente para garantir maior controle dos dados
Algumas imagens estão com tamanhos desproporcionais, e isso também será corrigido em melhorias futuras
Apesar desses pontos, o projeto já está exatamente como eu queria para acompanhar minha coleção no dia a dia 💙

## Tecnologia utilizada 🛠️

- HTML

- JavaScript
